# Star Trek Quiz

Disclaimer: This sample question is not intended to be a real quiz question, but rather a demonstration of how to use the question API to create a question with dynamic content. Although we tried to make the question as accurate as possible, they were generated with the help of AI and may contain inaccuracies.

The topic was chosen to be unrelated to any specific courses to avoid any leak of real quiz questions (and to make it more fun to read). Anyone who can get full points for the test gets my deepest respect and admiration.

The question groups demonstrate the following types of questions:

- [episodes](episodes) - simple numerical question with a single correct answer
- [names](names) - single (`enterprise.md`, `riker.md`) and multiple-choice (`picard.md`) questions
- [open](open) - open text question that is not automatically graded
- [technical](technical) - ordered multiple-choice question (an algorithm-like question)
- [warp](warp) - a dynamic question that generates numeric questions where the student needs to calculate the answer (random parameters are generated in the question code)
